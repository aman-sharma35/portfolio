Project 2

To-Do List

The to-do list web application allows users to create and manage a list of tasks they need to complete. The application will have a user-friendly interface with a form for entering new tasks, a list of current tasks with the ability to mark them as complete, and a delete button to remove completed tasks. Users can add new tasks to the list by entering the task name and description in the form provided. The application will then store the task in a database and display it in the task list. Users can mark a task as complete by clicking a checkbox next to the task, and the task will be visually differentiated from incomplete tasks.
